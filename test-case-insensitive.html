üîç What's Changed

  • webrtc.js: Added displayRoomName variable to track original capitalization
  • Firebase operations: Always use roomId.toLowerCase()
  • Display functions: Use displayRoomName for UI elements
  • Admin functions: Use lowercase for Firebase but show original in messages
  • URL handling: Store lowercase in URLs for consistency
  • localStorage: Preserve original capitalization for better UX

  • Admin Panel: Works correctly with both display and storage names
